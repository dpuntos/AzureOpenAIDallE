# AzureOpenAIDallE
An Azure function wich uses Azure OpenAI services and Dall路E integration.
## Description
This Azure Function allows you to send a description as a parameter in the URL. The result is an image generated by Dall路E using the indicated description.
<br><br>
For example:<br><h5>
>https://{resourceName}.azurewebsites.net/api/Function1?code={api_key}==&desc=a portrait of a ninja cat painted in Van Gogh style
</h5><br>
The function receives the description. Finally, the generated image by Dall路E is obtained through the integration with Azure OpenAI services.
